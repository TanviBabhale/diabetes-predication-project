
🩺 Diabetes Prediction App

This is a Machine Learning + Streamlit Web App that predicts whether a patient is likely to have Diabetes or not, based on medical details.

🚀 How It Works

User enters patient health details (Pregnancies, Glucose, Blood Pressure, BMI, Age, etc.).

The trained Random Forest Model processes the data.

The app predicts whether the patient is:

Likely to have Diabetes ⚠️

Unlikely to have Diabetes ✅
